COMMANDS/QUERY IN MYSQL

  1. CREATE

    2. The CREATE function is used to create a database. It is also used to create a table(Example:CREATE TABLE LISTOFJOBS(TASKID INT(3) PRIMARY KEY AUTO-INCREMENT,TASKNAME VARCHAR(255), TIME INT(3));

  2. DROP

    3. The DROP function is used to delete columns, tables and database(Example: DROP TABLE LISTOFJOBS).

  3. ALTER

    4. The ALTER function is used to change the columns of a table(Example: ALTER TABLE LISTOFJOBS DROP TIME).

  4. ADD

    5. The ADD function is used to add columns to a table.



  5. UPDATE

    6. The UPDATE function is used to make changes to the table data.



  6. SET

    7. The SET function is used with UPDATE to specify which columns or values are to be changed.



  7. INSERT

    8. The INSERT function is used to insert new values to the table.



  8. SELECT

    9. The SELECT statement in SQL is used to fetch or retrieve data from a database. It allows users to access the data and retrieve specific data based on specific conditions.
