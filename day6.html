COMMANDS/QUERY IN MYSQL

  1. How to create gmail account?
    1. Go to Google accounts sign-in page and click create account.
    2. Enter Name,date-of-birth and gender.
    3. Create your email address and put your desired password and confirm it.
    4. Put your phone number and verify it.
    5. You can add recovery options but it is not mandatory.
    6. Gmail account is created.


  2. How to create Github account?
    1. Go to Github.com and click sign up.
    2. Verify your email address.
    3. Create username and password.
    4. Click create account and choose how many team members
    5. Select the free plan or Github pro.


  3. How to connect VSCode to Github Account?
    1. Open Visual Studio Code
    2. Click on the Accounts icon in VSCode
    3. Click Sign in to github.
    4. Sign in to your github account.
    5. Click continue with Visual Studio Code
    6. Go to Source Control and click download git for desktop
    7. Setup Git and your VSCode and Github are connected.


  4. How to clone/pull repository from github?
    1. Go to GitHub.com and open your repository.
    2. Go to Code and click open with github desktop.
    3. Download github desktop and install it.
    4. Open github desktop and click on current repository
    5. Click Add and select the Clone option
    6. Select the location of an empty folder and clone the repository there.


  5. How to push repository to github?
    1. Open GitHub desktop
    2. Go to File in the menu bar.
    3. Click Add local repository
    4. Choose the local repository to be added.
    5. Click Add repository


  6. How to publish the repository as a live website?
    1. Open your repository on Github.com.
    2. Go to settings and click on pages option.
    3. Under Branch and deployment click deploy from a branch.
    4. Select the main branch and select the root folder while keeping the name of your main webpage as "index.html".
    5. Click save and reload.
    6. After some time, You can visit your live site.
