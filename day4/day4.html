Keys/Types of keys

  1. Super Keys

    2. All the values of a column that represent a particular row are known as super keys. It is the collection of all keys.

  2. Candidate Key

    3. All the values of a column that can be used to uniquely represent a particular row are called candidate keys

  3. Primary key

    4. The unique identifier that is the most feasible to use and is used to represent a particular row is called primary key.

  4. Alternate Key

    5. The unique identifiers besides the primary key that can be used to represent a particular row are called alternate keys

  5. Foreign key

    6. The unique identifier that links the table to the primary key of another table is called foreign key.

  6. Composite key

    7. The unique identifier made of one of the candidate keys and other values together is called a composite key

Normalization

Process of breaking down larrger tables into multiple smaller tables is called normalization. There are many levels of normalization including 1st level,2nd level,3rd level and so on.
